# WorkCentrik - Temporary Landing Page

A high-performance, fully optimized React landing page for [**WorkCentrik**](https://workcentrik.com/), a digital agency based in Cape Town, South Africa. This temporary digital storefront is designed to establish brand authority, capture leads via WhatsApp, and maintain top-tier SEO presence while the comprehensive agency website is under development.

## 🚀 Key Features

* **Perfect 100 Lighthouse Score:** Fully optimized for Performance, Accessibility (WCAG compliant), Best Practices, and SEO.
* **Mobile-First & Fully Responsive:** Fluid typography and layout adjustments ensure a flawless experience across all devices, including a targeted `100svh` hero section for modern mobile browsers.
* **Direct WhatsApp Integration:** Features high-conversion, dynamic WhatsApp click-to-chat links, including a persistent floating CTA.
* **Premium UI/UX:** Built with a custom CSS design system utilizing CSS variables, smooth scrolling, interactive hover states, and crisp vector iconography.
* **Top-Tier SEO:** Deeply integrated meta tags, Open Graph (OG) tags, and Twitter cards optimized for local search intent in the South African web design and development market.

## 🛠 Tech Stack

* **Framework:** React 18
* **Build Tool:** Vite
* **Styling:** Custom CSS3 (CSS Variables, Flexbox, CSS Grid)
* **Icons:** Lucide React
* **Hosting Context:** Configured for seamless deployment on Netlify.
