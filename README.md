# WorkCentrik | Digital Excellence 🚀

> **A premium, Lighthouse-100 optimized corporate website for WorkCentrik.** > Built with modern React and Vite, featuring bespoke UI design, flawless accessibility, and dynamic WhatsApp lead-generation routing. Proudly developed in Cape Town, South Africa.

---

## 🌟 Key Features & Architectural Standards

This project was built from the ground up to set a new standard in digital agency presentation, focusing on performance, UX, and technical SEO.

### 1. Elite Performance & Accessibility (Lighthouse 100)
* **WCAG AA Compliant:** High-contrast color palette (`#192a40` Navy and `#21b8a6` Teal) ensuring perfect legibility across all devices.
* **Semantic HTML & ARIA:** Properly structured tags and labels for screen readers.
* **Optimized Assets:** Zero heavy CSS frameworks. Pure, scalable CSS architecture for instant load times.

### 2. Premium UI/UX Engineering
* **Bespoke Interactions:** Smooth, cubic-bezier hover states, animated navigation underlines, and interactive "Most Popular" product cards.
* **Custom Micro-Aesthetics:** Branded text selection highlights and a custom-styled WebKit scrollbar to maintain total immersion.
* **Flawless Responsive Grid:** Advanced Flexbox and CSS Grid layouts ensure pixel-perfect alignment from 4k desktop monitors down to mobile screens.

### 3. Integrated Lead Generation
* **Dynamic WhatsApp Routing:** Each service card features a highly targeted, pre-filled WhatsApp link (e.g., *"Hi WorkCentrik! 👋 I'm interested in your E-commerce Solutions service..."*), removing all friction for potential clients.
* **Floating Action Button (FAB):** Persistent, animated WhatsApp FAB for instant contact at any point in the user journey.

### 4. Technical SEO & Social Previews
* **Robust Content Security Policy (CSP):** Hardened security headers allowing secure execution of local scripts and Google Fonts.
* **Open Graph & Twitter Cards:** Fully configured meta tags pointing to a custom, 1200x630px social sharing image (`workcentrik-og.png`) for stunning previews on LinkedIn, X, and WhatsApp.

## 🛠️ Tech Stack

* **Framework:** React 18
* **Build Tool:** Vite (for lightning-fast HMR and optimized production bundling)
* **Styling:** Pure CSS3 (Custom Properties, Flexbox, Grid)
* **Icons:** Lucide React
* **Typography:** 'Inter' (Google Fonts)
