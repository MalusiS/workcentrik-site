# WorkCentrik | Digital Excellence 🚀

> **A premium, Lighthouse-100 optimized corporate website for WorkCentrik.** > Built with modern React and Vite, featuring bespoke UI design, flawless accessibility, a custom Intersection Observer ScrollSpy, and dual-channel lead generation routing. Proudly developed in Cape Town, South Africa.

---

## 🌟 Key Features & Architectural Standards

This project was built from the ground up to set a new standard in digital agency presentation, focusing on performance, UX, and technical SEO.

### 1. Elite Performance & Accessibility (Lighthouse 100)
* **WCAG AA Compliant:** High-contrast color palette (`#192a40` Navy and `#21b8a6` Teal) ensuring perfect legibility across all devices.
* **Semantic HTML & ARIA:** Properly structured tags and labels for screen readers.
* **Optimized Assets:** Zero heavy CSS frameworks. Pure, scalable CSS architecture for instant load times.

### 2. Premium UI/UX Engineering
* **Immersive Hero Section:** Engineered with abstract CSS radial glows, animated status pills ("Accepting new projects"), and stunning text gradients to instantly establish a premium agency feel.
* **Bespoke Interactions:** Smooth cubic-bezier hover states, animated navigation underlines, and glassmorphic UI elements for secondary contact methods.
* **Custom Micro-Aesthetics:** Branded text selection highlights and a custom-styled WebKit scrollbar to maintain total immersion.
* **Flawless Responsive Grid:** Advanced Flexbox and CSS Grid layouts ensure pixel-perfect alignment—from the equal-height 'About' section columns down to the mobile-optimized, frosted-glass hamburger navigation overlay.

### 3. Advanced React Logic & State Management
* **Bulletproof ScrollSpy Navigation:** Utilizes a custom `IntersectionObserver` with a `useRef` tracking system evaluated in reverse DOM order, guaranteeing the navigation perfectly highlights the active section regardless of section height or scroll direction.
* **Expandable Content:** State-driven UI for expanding/collapsing the Executive Leadership profile within the Agency DNA section without disrupting the grid layout.

### 4. Dual-Channel Lead Generation
* **Split-Action Service Cards:** Users can choose their preferred communication method directly from the pricing grid.
* **Dynamic WhatsApp Routing:** Generates a highly targeted, pre-filled WhatsApp link (e.g., *"Hi WorkCentrik! 👋 I'm interested in your E-commerce Solutions service..."*).
* **Auto-Scrolling Email Hooks:** Clicking the email quote button triggers a smooth scroll to the contact section and automatically pre-selects the requested service in the dropdown menu.
* **Formspree Integration:** A fully functional, pixel-perfect, custom-styled contact form directly wired to a Formspree endpoint for immediate lead capture.
* **Floating Action Button (FAB):** Persistent, animated WhatsApp FAB for instant contact at any point in the user journey.

### 5. Technical SEO & Social Previews
* **Robust Content Security Policy (CSP):** Hardened security headers allowing secure execution of local scripts and Google Fonts.
* **Open Graph & Twitter Cards:** Fully configured meta tags pointing to a custom, 1200x630px social sharing image for stunning previews on LinkedIn, X, and WhatsApp.

## 🛠️ Tech Stack

* **Framework:** React 18
* **Build Tool:** Vite (for lightning-fast HMR and optimized production bundling)
* **Styling:** Pure CSS3 (Custom Properties, Flexbox, Grid, Glassmorphism)
* **Scroll Tracking:** Custom Intersection Observer API
* **Form Handling:** Formspree
* **Icons:** Lucide React
* **Typography:** 'Inter' (Google Fonts)
