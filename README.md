# WorkCentrik - Temporary Landing Page

A high-performance, fully optimized React landing page for **WorkCentrik**, a digital agency based in Cape Town, South Africa. This temporary digital storefront is designed to establish brand authority, capture leads via WhatsApp, and maintain top-tier SEO presence while the comprehensive agency website is under development.

## 🚀 Key Features

* **Perfect 100 Lighthouse Score:** Fully optimized for Performance, Accessibility (WCAG compliant), Best Practices, and SEO.
* **Mobile-First & Fully Responsive:** Fluid typography and layout adjustments ensure a flawless experience across all devices, including a targeted `100svh` hero section for modern mobile browsers.
* **Direct WhatsApp Integration:** Features high-conversion, dynamic WhatsApp click-to-chat links, including a persistent floating CTA.
* **Premium UI/UX:** Built with a custom CSS design system utilizing CSS variables, smooth scrolling, interactive hover states, and crisp vector iconography.
* **Top-Tier SEO:** Deeply integrated meta tags, Open Graph (OG) tags, and Twitter cards optimized for local search intent in the South African web design and development market.

## 🛠 Tech Stack

* **Framework:** React 18
* **Build Tool:** Vite
* **Styling:** Custom CSS3 (CSS Variables, Flexbox, CSS Grid)
* **Icons:** Lucide React
* **Hosting Context:** Configured for seamless deployment on Netlify.

## 💻 Getting Started

**Prerequisites**

Make sure you have `Node.js` installed on your machine.

**Installation**

1. Clone the repository:

   ```git clone <repository-url>

2. Navigate to the project directory:

   ```cd workcentrik-site

3. Install the dependencies:

   ```npm install

**Development**

To start the local Vite development server with Hot Module Replacement (HMR):

```npm run dev

The site will be available at `http://localhost:5173`.

## 📦 Deployment

This project is optimized for rapid deployment to `Netlify`.

1. Run the build command to generate production-ready static files:

   ```npm run build

2. This will create a `dist/` directory in the root of your project.

3. Log in to `Netlify` and drag and drop the `dist/` folder into the manual deploy area.

4. Update DNS settings to point your domain name to your new Netlify deployment.
